---- split(create)----

:{height} split {new(file name)}
:{width} vsplit {new(file_name)}

new is used for create new file when (v)split

---- window size ----
height:
	{height} CTRL-w _		set
	{height} CTRL-w -		delta
	{height} CTRL-w +		delta

	:res[size] -{height}	delta
	:res[size] +{height}	delta

width:
	{width} CTRL-w |		set
	{width} CTRL-w < 		delta
	{width} CTRL_w > 		delta	

	:virtical res[size] +{width}
	:virtical res[size] -{width}
